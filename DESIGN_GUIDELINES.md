## ⭐ 핵심 규칙

### 🔲 1타일 규칙 (매우 중요!)

> **모든 캐릭터와 몬스터는 시각적 크기와 관계없이 논리적으로 1타일(1칸)만 차지합니다.**

```
예시:
┌─────────────────────────────────────────┐
│  슬라임 (32x32px) → 1타일 차지          │
│  전사 (64x64px)   → 1타일 차지          │
│  드래곤 (128x128px) → 1타일 차지        │
└─────────────────────────────────────────┘
```

**적용 범위:**
- ✅ 충돌 판정: 타일 중심점 기준
- ✅ 이동 경로: 1칸 단위
- ✅ 타겟팅: 해당 타일 선택
- ✅ 스폰 위치: 1타일에 1엔티티

**시각적 렌더링:**
- 큰 스프라이트는 타일 중심에서 위로 확장
- 앵커 포인트는 스프라이트 하단 중앙

## 1. 비주얼 스타일: 다크 판타지 (90s RPG)

### 시점
- **필수**: 아이소메트릭 2.5D (대각선 ~30° 각도)
- 캐릭터와 몬스터는 대각선 방향을 향함
- 타일은 다이아몬드 형태 (64x48 픽셀)

### 분위기
- **무드**: 어둡고, 우울하고, 무겁고, 중세적이고, 억압적인
- **분위기**: 신비롭고, 위험하고, 저주받은 땅

### 색상 팔레트

```
┌────────────────────────────────────────────────────┐
│  기본/어둠     │ #0a0a0a  #1a1a2e  #16213e        │
│  피/경고       │ #8b0000  #660000  #cc0000        │
│  마법/신비     │ #4a0080  #2d1b4e  #1a0a2e        │
│  신성/희망     │ #8b7355  #c5c6c7 (희미하게)      │
│  독/저주       │ #2d4a22  #3d1a4f                 │
└────────────────────────────────────────────────────┘
```

### 금지 스타일 ❌
- "귀여운" 또는 "치비" 비율
- 밝고 행복한 파스텔 색상
- 변형된 큰 머리 (공포스러운 경우 제외)
- 친근한 동물 (토끼, 다람쥐) - 언데드/변이 버전 사용
- 만화/카툰 스타일

## 2. 스프라이트시트 규격

### 스프라이트시트 레이아웃 (필수)

```
        Col 0    Col 1    Col 2    Col 3    Col 4    Col 5
       ┌────────┬────────┬────────┬────────┬────────┬────────┐
Row 0  │ IDLE 0 │ IDLE 1 │ IDLE 2 │ IDLE 3 │        │        │  (4프레임)
       ├────────┼────────┼────────┼────────┼────────┼────────┤
Row 1  │ MOVE 0 │ MOVE 1 │ MOVE 2 │ MOVE 3 │        │        │  (4프레임)
       ├────────┼────────┼────────┼────────┼────────┼────────┤
Row 2  │ ATK 0  │ ATK 1  │ ATK 2  │ ATK 3  │ ATK 4  │ ATK 5  │  (6프레임)
       ├────────┼────────┼────────┼────────┼────────┼────────┤
Row 3  │ DEATH0 │ DEATH1 │ DEATH2 │ DEATH3 │        │        │  (4프레임)
       └────────┴────────┴────────┴────────┴────────┴────────┘
```

### 프레임 크기

| 엔티티 타입 | 프레임 크기 | 시트 크기 (예상) |
|-------------|-------------|------------------|
| 캐릭터 (모든 클래스) | 64x64 px | 384x256 px |
| 소형 몬스터 (Lv 1-10) | 32x32 px | 192x128 px |
| 중형 몬스터 (Lv 11-50) | 48x48 px | 288x192 px |
| 대형 몬스터 (Lv 51+) | 64x64 px | 384x256 px |
| 보스 | 128x128 px | 768x512 px |
| 스킬 이펙트 | 64x64 px | 384x64 px (1행) |

## 3. 애니메이션 요구사항

### 표준 요구사항

| 애니메이션 | 최소 프레임 | 총 시간 | 프레임당 시간 | 비고 |
|-----------|------------|---------|--------------|------|
| Idle (대기) | 4 프레임 | 600ms | 150ms | 미세한 호흡 |
| Move (이동) | 4 프레임 | 400ms | 100ms | 자연스러운 걷기 |
| Attack (공격) | 6 프레임 | 500ms | ~83ms | 다이나믹, 임팩트 |
| Death (사망) | 4 프레임 | 600ms | 150ms | 쓰러져서 정지 |
| Skill (스킬) | 6+ 프레임 | 800ms | ~133ms | 화려하고 강렬 |

### 공격 애니메이션 가이드라인
- **쿨다운**: 최소 1000ms
- **전사**: 무거운 베기, 방패 강타
- **마법사**: 지팡이 들기 + 마법 폭발
- **도적**: 빠른 찌르기, 백스탭
- **성직자**: 신성한 타격, 신성한 빛

### 스킬/마법 이펙트
- 화염, 얼음, 번개는 **화면 흔들림** 수준의 임팩트
- 모든 스펠 이펙트는 **최소 6프레임**
- 고대비 사용 (어두운 배경에 밝은 효과)
- 시전 파티클 + 타격 파티클 포함

## 4. 캐릭터 & 몬스터

### 캐릭터 디자인
- **비율**: 성인 신체 비율 (7-8등신)
- **스타일**: 약간 양식화되었지만 치비 아님

#### 클래스별

| 클래스 | 남성 외형 | 여성 외형 |
|--------|----------|----------|
| 전사 (Warrior) | 무거운 판금 갑옷, 전투로 닳은 | 어두운 강철 비키니 아머, 숄더 |
| 마법사 (Mage) | 어두운 로브, 후드, 빛나는 눈 | 노출이 있는 어두운 로브 |
| 도적 (Rogue) | 가벼운 가죽, 후드 망토 | 타이트한 가죽, 암살자 미학 |
| 성직자 (Cleric) | 낡은 성스러운 의복 | 우아하지만 어두운 사제 로브 |
| 무도가 (Martial Artist) | 맨가슴, 붕대, 흉터 | 타이트한 전투복 |

### 레벨별 몬스터 디자인

| 레벨 범위 | 몬스터 타입 | 시각적 스타일 |
|----------|------------|--------------|
| 1-10 | 쥐, 박쥐, 슬라임, 뱀 | 작은 해충 |
| 11-25 | 늑대, 스켈레톤, 고블린 | 인간형 위협 |
| 26-50 | 유령, 늑대인간, 악마 | 초자연적 공포 |
| 51-75 | 리치, 다크 나이트, 서큐버스 | 강력한 엘리트 |
| 76-99 | 드래곤, 고대의 악 | 보스급 위협 |

## 5. 오디오

### BGM

| 장면 | 스타일 | 볼륨 |
|------|--------|------|
| 메인 메뉴 | 불길한 합창, 저음 드론 | 50% |
| 마을 | 우울하고 평화로운 어둠 | 40% |
| 던전 | 긴장감, 에코, 먼 위협 | 50% |
| 보스전 | 웅장하고 격렬한 오케스트라 | 60% |

### 효과음

| 카테고리 | 사운드 타입 | 스타일 |
|----------|-----------|--------|
| 무기 휘두르기 | 우쉬 | 빠르고 날카롭게 |
| 타격 임팩트 | 살: 둔탁 / 금속: 쨍 | 묵직하고 만족스럽게 |
| 스펠 시전 | 원소별 휘씉 | 신비롭고 고조 |
| 스펠 임팩트 | 폭발, 쨍, 지글 | 강력하고 울림 |
| 환경음 | 우울한 바람, 던전 에코 | 분위기 있게 |

## 6. 파일 네이밍 규칙

### 스프라이트시트

```
캐릭터:
/assets/characters/{class}/{gender}_spritesheet.png
예: /assets/characters/warrior/male_spritesheet.png

몬스터:
/assets/monsters/{type}/spritesheet.png
예: /assets/monsters/slime/spritesheet.png

스킬 이펙트:
/assets/skills/{class}/{skill}_fx.png
예: /assets/skills/mage/fireball_fx.png
```

### 오디오

```
BGM:
/assets/audio/bgm/{scene}.mp3
예: /assets/audio/bgm/village.mp3

SFX:
/assets/audio/sfx/{category}/{sound}.mp3
예: /assets/audio/sfx/attack/sword_swing.mp3
```
